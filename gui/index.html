Why KWT5H13LD

Built for Defenders

Designed by security practitioners for security practitioners. No fluff — just the tools you need.

Zero Dependencies

Runs on pure bash. Uses system tools already on your servers. No Python, no Ruby, no Node — just shell.

Modular Architecture

Load only what you need. Each module is self-contained and independently executable via CLI or interactive menu.

Multi-Cloud Ready

Supports AWS, Azure, and GCP with native CLI integration for cloud security auditing and IAM review.

Automated Reports

Interactive + CLI

Get Started

Installation

Up and running in under 30 seconds.

1
Clone the repository git clone https://github.com/SiteQ8/KWT5h13ld.git
2
Make it executable cd KWT5h13ld && chmod +x kwt5h13ld.sh
3
Run the toolkit sudo ./kwt5h13ld.sh
→
Or run a specific module directly ./kwt5h13ld.sh harden-linux -v
